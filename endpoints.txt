API Endpoints

GET /commands
	List all known commands
	Response format
		[
			{
				name:"Example 1",
				description:"Optional description",
				windows:1,
				args:[
					{"name":"arg1", type:"enum", options:["foo", "bar"]},
					{"name":"stringarg", type:"string", hint:"your hint"},
					...
				]
			},
			...
		]

GET /layouts
	List all known layouts
	Response format
		[
			{
				name:"Fullscreen",
				frames:[
					{id:1, x:0, y:0, w:300, h:400, screen:4},
					...
				],
				screens:[
					{id:5, w:1920, h:1080},
					...
				]
			},
			...
		]

GET/POST /status
	Get status information on the daemon, such as the
	current layout and running commands
	Response format:
		{
			layout:"name",
			running:[
				"command1",
				...
			]
		}

GET/POST /reset
	Restore initial state, including layout

GET/POST /stop/name
	Stop a running command

GET/POST /layout/name
	Activate a layout

POST /command/name
	Start a command
	Request format
		{
			frame:4,
			fullscreen:1,
			arguments:{
				"arg1":"value1",
				...
			}
		}
