API Endpoints

GET /commands
	List all known commands
	Response format
		[
			{
				name:"Example 1",
				args:[
					{"name":"arg1", type:"enum", options:["foo", "bar"]},
					{"name":"stringarg", type:"string", hint:"your hint"},
					...
				]
			},
			...
		]

GET /layouts
	List all known layouts
	Response format
		[
			{
				name:"Fullscreen",
				xres:800,
				yres:600,
				frames:[
					{id:1, x:0, y:0, w:300, h:400},
					...
				]
			},
			...
		]

GET/POST /status
	Get status information on the daemon, such as the
	current layout and running commands

GET/POST /reset
	Restore initial state, including layout

GET/POST /stop/name
	Stop a running command

GET/POST /layout/name
	Activate a layout

POST /command/name
	Start a command
	Request format
		{
			fullscreen:1,
			arguments:[
				{name:"arg1", value:"foo"},
				...
			]
		}
